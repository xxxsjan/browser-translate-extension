# 浏览器插件

## 技术栈

plasmo + React + typescript+ tailwindcss + daisyui

## 环境

python3.6+

## 使用

- 安装依赖 pnpm i

- 打包项目 npm run build

- 浏览器进入扩展程序管理，点击加载已解压，开发选 build/chrome-mv3-dev，生产选择文件夹 build/chrome-mv3-prod
